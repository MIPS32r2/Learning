Quando precisamos determinar uma posição específica dentro de uma Expressão Regular, podemos usar os Metacaracteres *Âncoras* para isso.

- para delimitar que precisamos que um determinada expressão regular comece com um caractere específico, usamos o acento circunflexo *^*.

![[Pasted image 20260219022805_1.png]]

- Agora quando precisamos definir que a expressão regular termine com um determinado padrão, usamos o *$*.

![[Pasted image 20260219023158.png]]

Obs: no exemplo acima usamos as aspas simples, como o shell interpreta por padrão que o símbolo de *$* representa uma expansão, colocamos dessa maneira para que o shell não interpretasse assim.
