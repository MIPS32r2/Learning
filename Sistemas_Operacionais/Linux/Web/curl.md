
o curl (client URL) é uma ferramenta que nos permite mandar requisições web assim como um web browser qualquer.

-s = modo silencioso
-O = download
-k = ignora erros de verificação de SSL
-I = mostra somente os cabeçalhos da resposta
-i = mostra o cabeçalho e o corpo da resposta
-A = define o user-agent
-v = mostra os headers de requisição
-u = define o usuário e senha
-H = define múltiplos cabeçalhos
-L = segue o redirecionamento de uma página
-X = define o método HTTP a ser usado na requisição
-b = serve para enviar cookies junto da requisição ou ler cookies de um arquivo24
